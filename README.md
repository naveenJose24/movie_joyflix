# JoyFlix

Static web app for browsing movies/TV via TMDB, streamed through tmdbplayer.nunesnetwork.com.

## Structure

- `index.html`, `app.js`, `style.css` — the app
- `public/` — static assets
- `vercel.json` — static deploy config

## Run locally

Serve the root directory with any static file server, e.g.:

```bash
npx serve .
```
